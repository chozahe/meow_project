# Игра Виселица

Простая игра "Виселица". Сохраняет состояние игры в 'localStorage`.

## Особенности:
- Новая игра с случайным словом, полученным из RandomWordAPI
- Отслеживание жизней
- Виртуальная клавиатура
- Сохранение в `localStorage`

## Используемые технологии:
- HTML
- CSS (Bulma)
- JavaScript (Native)
